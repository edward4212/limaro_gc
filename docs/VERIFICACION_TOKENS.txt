```bash

```

Esto creará la tabla `archivo_acceso_temporal` con:
- Índices en token y expira_en
- Foreign keys a archivo y usuario
- Timestamp automático de creación

### 2. Configurar Limpieza de Tokens Expirados
```bash
# Agregar a crontab (ejecutar cada hora):
0 * * * * mysql -u root -pPASSWORD limarocloud_limaro -e \
  "DELETE FROM archivo_acceso_temporal WHERE expira_en <= NOW();"
```

## 🔍 Verificación de Seguridad

### Token Generation ✅
- Usando `random_bytes(32)` - Criptográficamente seguro
- Convertido a hexadecimal (64 caracteres)
- Único por descarga

### Token Validation ✅
- Stored en BD con FK a usuario que lo generó
- Expiración configurable (1 hora)
- Validated con prepared statements
- Logging de intentos fallidos

### Access Control ✅
- Sin autenticación requerida (token en lugar de sesión)
- Perfecto para Microsoft Office Viewer
- Path traversal prevention en ArchivoModel
- MIME type validation
